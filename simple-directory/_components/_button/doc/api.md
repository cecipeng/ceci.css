## API
| 参数     | 说明           |
|----------|----------------|
| ui-button--clicked | 添加点击特效,移动端默认添加 |
| ui-button--disabled | 不可点击态 |

按钮类型

| 参数     | 说明           |
|----------|----------------|
| ui-button__main | 主要按钮 |
| ui-button__default | 默认按钮 |

按钮尺寸
（不设定时默认中号）

| 参数     | 说明           |
|----------|----------------|
| ui-button--l | 大号 |
| ui-button--s | 小号 |

按钮形态
（不设定时默认圆角矩形，多种形态可组合使用）

| 参数     | 说明           |
|----------|----------------|
| ui-button__circle | 圆形按钮 |
| ui-button__ghost | 幽灵按钮 |
