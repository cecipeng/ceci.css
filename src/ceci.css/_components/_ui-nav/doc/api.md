## API

| 参数     | 说明           | 添加位置     |
|----------|----------------|----------|
| ui-nav--horizontal    | 【形态】水平导航 | ui-nav   |
| ui-nav--vertical    | 【形态】垂直导航（子菜单浮动） | ui-nav   |
| ui-nav--inline    | 【形态】垂直导航（子菜单内联） | ui-nav   |
| ui-nav__item--active    | 激活状态 | ui-nav__item   |
| ui-nav__item--open    | 子菜单展开。仅在垂直导航（内联）有效 | ui-nav__item   |
